Project Homepage

Granfluencers in Digital Media

A bilingual project site for the computational study of older social media creators, their content strategies, engagement patterns, and audience response.

Site Scope

One homepage, multiple research workflows

This homepage links the living project wiki, the collection workflow, the analysis interface, and the results layer. Every page has a matching Chinese counterpart, and the published GitHub Pages site is built directly from the HTML inside docs/.

Canonical dashboard data lives in data/dashboard/. The published site reads the synchronized mirror under docs/data/.

Quick Actions
  • Use Wiki for the project map and rules.
  • Use Collect for TikHub and export workflow details.
  • Use Analysis for site-facing data views.
  • Use Results to stage findings and writeups.

How the docs site fits the repository

  1. Collect and normalize account data through src/collect/.
  2. Export canonical dashboard JSON into data/dashboard/.
  3. Sync the published copy into docs/data/.
  4. Ship the HTML site from docs/ to GitHub Pages.
