# Granfluencers in Digital Media

This is the project documentation site for **Granfluencers in Digital Media: A Computational Analysis of Content, Engagement, and User Response**.

## Project Overview

This project examines how older social media creators ("granfluencers") produce content, attract engagement, and shape user response on digital platforms. The research combines computational social science, digital communication, and advertising analysis.

## Research Scope

- Granfluencer profiles and account characteristics
- Posts, captions, hashtags, and posting time
- Images and detailed visual descriptions
- Engagement metrics such as likes, comments, and shares
- Creator reply behavior
- Advertising, products, brands, and commercial content
- User comments and audience response

## Workflow

1. Data collection
2. Data preprocessing
3. Computational coding and content analysis
4. Statistical and interpretive analysis
5. Research findings and theoretical discussion
