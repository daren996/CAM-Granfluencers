# 数字媒体中的老年网红

这是 **Granfluencers in Digital Media: A Computational Analysis of Content, Engagement, and User Response** 项目的文档主页。

## 项目概览

本项目关注老年社交媒体创作者（granfluencers）如何生产内容、获得互动并影响用户回应。研究结合计算社会科学、数字传播与广告分析方法展开。

## 研究范围

- Granfluencer 账号与人物特征
- 帖子、文案、标签与发布时间
- 图片及其细化视觉描述
- 点赞、评论、分享等互动指标
- 创作者回复行为
- 广告、产品、品牌与商业化内容
- 用户评论与受众反应

## 工作流程

1. 数据收集
2. 数据预处理
3. 计算编码与内容分析
4. 统计分析与解释分析
5. 研究发现与理论讨论
