项目主页

数字媒体中的老年网红

这是一个双语项目站点，用于组织关于老年社交媒体创作者的计算研究，包括内容策略、互动模式与受众回应。

站点定位

一个主页，连接多条研究工作流

当前首页连接项目 Wiki、数据收集、分析页面和结果页面。每个页面都保留英文与中文两个版本，GitHub Pages 直接发布 docs/ 中的 HTML 文件。

项目的规范数据目录位于 data/dashboard/，发布站点读取的是同步到 docs/data/ 的镜像文件。

快速入口
  • 用 Wiki 查看项目结构与规则。
  • 用 收集 页面管理 TikHub 工作流。
  • 用 分析 页面查看已同步的数据。
  • 用 结果 页面整理研究输出。

工作流

文档站如何连接仓库

  1. 通过 src/collect/ 完成账号数据采集与规范化。
  2. 把规范数据导出到 data/dashboard/。
  3. 把发布所需副本同步到 docs/data/。
  4. 由 GitHub Pages 直接发布 docs/ 中的 HTML 站点。
python3 -m src.collect export-dashboard --input data/collect
python3 -m src.collect sync-docs-data
make docs-data
